🚀 ML Foundations Visualizer

An interactive web application to visualize core Machine Learning concepts like Vector Similarity, Functions & Derivatives, and Gradient Descent using Python and Streamlit.

---

📌 Objective

This project helps users understand important mathematical concepts used in Machine Learning through interactive visualizations and computations.

---

🧩 Features

🔹 Vector Similarity

- Input two vectors
- Compute:
  - Dot Product
  - Cosine Similarity
  - Euclidean Distance
- Visualize vectors in 2D space

---

🔹 Function & Derivative Visualizer

- Enter mathematical functions (e.g., "x**2", "sin(x)")
- Compute derivatives
- Plot function and its derivative

---

🔹 Gradient Descent Simulation

- Choose starting value
- Adjust learning rate and iterations
- Visualize optimization path
- Understand convergence behavior

---

🛠️ Tech Stack

- Python
- NumPy
- Matplotlib
- SymPy
- Streamlit

---

⚙️ Installation

1. Clone the repository:

git clone https://github.com/your-username/ml-visualizer.git
cd ml-visualizer

2. Install dependencies:

pip install -r requirements.txt

---

▶️ Run the Application

streamlit run app.py

Then open in browser:

http://localhost:8501
